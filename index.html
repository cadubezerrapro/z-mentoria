Sobre a mentoria

Preciso já ter um negócio funcionando? +

Não necessariamente. Mas você precisa ter algo para vender — conhecimento, serviço ou produto digital. A mentoria entrega a estrutura de tráfego e vendas. O que vender é com você.

É um curso ou uma mentoria? +

Mentoria. Você não só assiste vídeos — tem acesso direto a mim nas calls mensais e na comunidade privada. Você traz o problema real e eu mostro o caminho. Nada de suporte por ticket ou bot.

Quanto tempo leva para ver resultado? +

O método foi desenhado para gerar as primeiras vendas em até 30 dias. Pressupõe execução — quem só assiste e não implementa não vai ter resultado. Quem executa, tem. Simples assim.

Como funciona receber em euro/dólar? +

Você aprende a estruturar tráfego para mercados externos (Europa e EUA) e receber internacionalmente sem abrir empresa fora. Você opera do Brasil — com os gateways e contas certas que eu uso.

Tem suporte entre as calls? +

Sim. A comunidade privada é o canal direto entre as calls. Você posta dúvidas, criativos, métricas — e recebe feedback meu e dos outros mentorados que estão executando.

Qual o investimento? +

Discuto na conversa direta. Não publico valor aqui porque a mentoria é aplicada — passo por um filtro antes de aceitar. Me chama no WhatsApp e conversamos.

Sobre vibecoding e IA

Preciso saber programar para vibecoding? +

Não. Se você consegue descrever o que quer em português, consegue construir. O Claude Code escreve, testa e corrige o código — você direciona e aprende vendo acontecer.

Preciso pagar a API da Anthropic? +

O Claude Code tem plano pago (alguns dólares por mês para uso normal). Aprende a otimizar o uso. Se você usa para cobrar de clientes, o custo é repassado facilmente no preço do projeto.

Posso cobrar para fazer isso para clientes? +

Sim — esse é um dos objetivos. Apps simples (landing page com backend, bot de WhatsApp, painel interno) saem entre R$500 e R$5.000 dependendo do escopo. Você aprende a montar e entregar.

Funciona no Windows e no Mac? +

Sim. O Claude Code roda nos dois sistemas. O setup é diferente em cada um — você aprende a configurar o ambiente correto para o seu OS do zero.
